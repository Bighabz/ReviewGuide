### Root cause for bugs #1-#5 + #11

Positional zip in `backend/mcp_server/tools/product_affiliate.py:174-175`. The LLM-generated product list and the curated-links list are joined by index, not by name. Whatever order the LLM returns, slot N always gets curated entry N. Verified across two independent sessions producing different products but the **same** wrong-link assignments per slot.

### Root cause for bug #8

`frontend/app/chat/page.tsx:25-83` — `useEffect` has deps `[searchParams, router]` and internally calls `router.replace('/chat')` which mutates searchParams → re-triggers effect. Real bug, but does not cause a hard browser hang in current build — only amplifies slowness during a long LLM stream.
